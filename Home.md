# Employee Staffing Services - Full Stack Application

Welcome to the EPM-STAFFING-SERVICES wiki!

This is a comprehensive full-stack job portal and staffing management application built with modern technologies.

## Project Overview

The Employee Staffing Services application is a complete solution for companies to manage their hiring process, from posting job openings to tracking candidates through the entire recruitment pipeline.

## Technology Stack

### Frontend (React)

- **React 19** - Modern React framework
- **Vite** - Fast build tool and dev server
- **Tailwind CSS** - Utility-first CSS framework
- **React Router DOM** - Client-side routing
- **React Toastify** - User notifications
- **Framer Motion** - Animations
- **Axios** - HTTP client

### Backend (Node.js/Express)

- **Node.js** - JavaScript runtime
- **Express.js** - Web application framework
- **PostgreSQL** - Database management
- **bcrypt** - Password hashing
- **express-session** - Session management
- **nodemailer** - Email functionality

## Key Features

- **User Authentication**: OTP-based email verification system
- **Job Management**: Create, edit, and manage job postings
- **Candidate Tracking**: Complete pipeline management
- **Company Management**: Multi-company support with admin oversight
- **Responsive Design**: Mobile-first approach
- **Real-time Updates**: Live data synchronization

## Getting Started

1. Install dependencies: `npm run install:all`
2. Start development: `npm run dev`
3. Access frontend: `http://localhost:5173`
4. Access API: `http://localhost:4000`
